Progressive Web App

This webpage uses a service worker which makes it a progressive web app. It means it can use the features of a device and browser that enhance the experience for the user. First of all its easier to find then a regular application because its on a search engine. It still works like an application and can even work offline. To update the web app all the user has to do is connect to wifi. A PWA must be hosted on HTTPS for the safety and security of the user. All in all a progressive web app is reponsive, safe, and independent.
